# 仕様書: UI・多言語対応

## 機能概要
Webアプリ（ローカルサーバー + ブラウザ）として動作するUI。日本語・英語の切り替えに対応。

## 技術方針
- MVPはWebアプリ（ローカルサーバー + ブラウザ）で実装
- 必要に応じてElectron/Tauriでラップする拡張を見据える
- クロスプラットフォーム（Mac / Windows）

## 画面構成

### 画面1: メンバー管理画面
- メンバー一覧表示（有効/全メンバー切り替え）
- メンバー追加フォーム（氏名、性別、言語、担当区分、同性制限、配偶者、参加可能日）
- メンバー編集・無効化

### 画面2: スケジュール設定画面
- カレンダー形式で日曜日を表示
- 除外日の設定/解除（クリックでトグル）
- 年月の切り替え

### 画面3: 割り当て結果画面
- 月間の割り当て結果をカレンダー or リスト形式で表示
- 各日の2グループ×2人の構成を表示
- 制約違反の警告表示
- 手動調整機能（メンバーの差し替え）
- 自動生成ボタン
- エクスポート機能（CSV、LINE用テキスト）

### 共通
- 日本語/英語の言語切り替えトグル
- 年度選択

## 多言語対応
- UIラベル、メッセージ、ボタンテキスト等を国際化
- メンバー名は翻訳しない
- i18nライブラリを使用（具体的なライブラリは実装計画時に決定）

## 受け入れ基準
- [ ] ブラウザでローカルにアクセスして使用できる
- [ ] Mac / Windows の両方で動作する
- [ ] 3画面（メンバー管理、スケジュール設定、割り当て結果）が存在する
- [ ] 日本語/英語を切り替えられる
- [ ] 切り替え後、全UIテキストが指定言語で表示される
