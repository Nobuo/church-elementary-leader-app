# 仕様書: エクスポート・共有機能

## 機能概要
生成された割り当てスケジュールをCSV出力、およびLINE共有用のまとめ文章を生成する機能。

## ユースケース

### UC-1: CSVエクスポート
- **入力**: 対象年月（または期間）
- **出力**: CSV形式のファイル
- **CSV列**: 日付, グループ番号, メンバー1名前, メンバー1言語, メンバー2名前, メンバー2言語
- **文字コード**: UTF-8（BOM付き、Excel対応）

### UC-2: LINE用まとめ文章生成
- **入力**: 対象年月
- **出力**: LINEメッセージとしてコピペできるテキスト
- **フォーマット例**:
```
📅 2026年4月 リーダー担当表

4/5（日）
  グループ1: 田中さん・鈴木さん
  グループ2: 佐藤さん・山田さん

4/12（日）
  グループ1: 高橋さん・伊藤さん
  グループ2: 渡辺さん・中村さん

...
```
- **注意**: テキストはクリップボードにコピーできるようにUI上にコピーボタンを設置

### UC-3: 多言語対応
- **入力**: 出力言語（日本語 / 英語）
- **出力**: 指定言語でのCSV・LINE文章
- **注意**: メンバー名はそのまま出力（翻訳しない）。ラベルやヘッダーのみ切り替え

## 受け入れ基準
- [ ] 月単位でCSVエクスポートできる
- [ ] CSVがExcelで文字化けしない（UTF-8 BOM付き）
- [ ] LINE用のまとめ文章が生成される
- [ ] まとめ文章をワンクリックでコピーできる
- [ ] 日本語・英語で出力を切り替えられる
