# タスク 032: フロントエンドの合同日3人表示対応

## 概要
割り当て画面の表示・差し替えUIを、合同日の3人×1グループに対応させる。

## 依存タスク
- 030（ユースケースの合同日対応）

## 対象ファイル
- `src/presentation/views/` — 割り当て表示コンポーネント
- `src/presentation/controllers/` — APIコントローラ
- `src/presentation/i18n/` — i18nキー追加（必要に応じて）

## 実装手順

### Step 1: 割り当て表示の更新

1. 合同日は1グループ（3人）のみ表示
2. 分級日は2グループ（各2人）を表示（既存）
3. 合同日のグループラベルを適切に表示（「グループ1」ではなく「リーダー」など）

### Step 2: 差し替えUIの更新

1. 合同日の差し替え候補ドロップダウンが正しく動作すること
2. 3人のうちどの1人でも差し替え可能であること
3. 差し替え後の制約チェック結果が表示されること

### Step 3: 差し替え候補APIの更新

- コントローラで差し替え候補を返す際、合同日は全メンバーを候補とする（UPPER/LOWER制限なし）
- イベント日のHELPER除外は維持

### Step 4: i18n

- 必要に応じて「合同日」「混合グループ」などのキーを追加

## テスト方針

- 合同日の画面で3人×1グループが表示されること
- 分級日の画面で2人×2グループが表示されること（リグレッション）
- 差し替えが3人 Assignment で動作すること

## 完了条件
- [x] 合同日の割り当てが3人×1グループで表示される
- [x] 差し替えUIが3人に対応している
- [x] 分級日の既存表示が維持される
- [x] i18nキーが日英両方に追加されている
