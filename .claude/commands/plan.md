# Plan: 実装計画作成

仕様書から具体的な実装計画を立て、`tasks/` 配下にタスクとして保存してください。

## 手順
1. 対象の仕様書を `specs/` から読み込む
2. `.claude/architecture.md` と `.claude/coding-rules.md` を確認する
3. 実装タスクを分解し、依存関係を整理する
4. `tasks/` にタスクファイルとして保存する

## タスクに含める内容
- タスク概要
- 対象ファイル・モジュール
- 実装手順（ステップバイステップ）
- 依存タスク
- テスト方針
- 完了条件

## 出力形式
- ファイル名: `tasks/<task-id>_<summary>.md`
- タスクIDは連番（例: 001, 002）
