# Implement: 実装

タスクに基づいてコードを実装してください。

## 手順
1. 対象タスクを `tasks/` から読み込む
2. 関連する仕様書を `specs/` から確認する
3. `.claude/architecture.md` と `.claude/coding-rules.md` に従って実装する
4. テストを `tests/` に作成する
5. 実装後、タスクファイルのステータスを更新する

## 実装ルール
- `.claude/coding-rules.md` のコーディング規約に従うこと
- `.claude/architecture.md` のアーキテクチャに従うこと
- テストファーストで進めること
- 1タスク1コミット単位を目安にすること

## 完了条件
- コードが実装されている
- テストが通る
- タスクファイルのステータスが「完了」に更新されている
